# 📝 Message Crafter

## תיאור
יוצר הודעות מותאמות אישית ברמת hyper-personalization עם 45%+ response rate.

## רמות התאמה אישית
1. **בסיסית (15-20% תגובה)**
   - שם, חברה, תפקיד
   
2. **מתקדמת (25-35% תגובה)**
   - ניסיון, כישורים, תחומי עניין
   
3. **היפר (45-65% תגובה)**
   - אישיות, ערכים, סגנון תקשורת, micro-interests

## סוגי הודעות
- **Initial Outreach**: 50+ תבניות בעברית ואנגלית
- **Follow-up**: 5 רצפים חכמים
- **Nurture**: תוכן ערך לטווח ארוך

## נוסחאות מנצחות
```
Hook (וו) + Context (הקשר) + Value (ערך) + CTA (קריאה לפעולה) = 45%+ תגובה
```

### דוגמאות Hooks מנצחים:
- "שמתי לב ל[הישג ספציפי]..."
- "הקשר המשותף שלנו עם [שם]..."
- "[חברה] + [כישור שלך] = ?"
- "ראיתי את הפוסט שלך על [נושא]..."

## A/B Testing אוטומטי
- בדיקת subject lines
- אורך הודעה
- סגנון (פורמלי/חברי)
- עומק personalization

## טיפים להצלחה
1. **קצר וחד**: מקסימום 150 מילים
2. **אישי**: לפחות 3 נקודות אישיות
3. **ערך ברור**: מה המועמד ירוויח
4. **CTA אחד**: בקשה אחת ברורה

## שילוב שפות
- **עברית למגע ראשון**: +15% תגובה
- **מעבר חלק לאנגלית**: לפי הקונטקסט
- **Slang ישראלי**: "יאללה", "סבבה" במינון

## מדדים
- **זמן כתיבה**: 2-3 שניות להודעה
- **ציון איכות**: 85%+ בממוצע
- **Variants**: 3-5 גרסאות לכל הודעה
